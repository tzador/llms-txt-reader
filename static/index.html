llms-txt-reader.org


A human-friendly viewer for machine-targeted llms.txt files and their linked markdown files.

Usage: Simply prepend https://llms-txt-reader.org/ to any llms.txt, llms-full.txt, or *.md URL.

https://llms-txt-reader.org/

Examples:

  • https://llms-txt-reader.org/https://fastht.ml/docs/llms.txt
  • https://llms-txt-reader.org/https://www.better-auth.com/llms.txt
  • https://llms-txt-reader.org/https://svelte.dev/llms-full.txt
  • https://llms-txt-reader.org/https://llmstxt.org/llms.txt
  • Find more awesome llms.txt files here.
